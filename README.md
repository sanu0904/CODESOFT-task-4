📌 CODESOFT-task-4

A movie recommendation system that suggests movies of the same genre to the user.

It uses a predefined list of movies, and based on the genres of the movie you select, the system recommends similar movies from that list. The project is simple to understand and is a great starting point for learning how recommendation/suggestion systems work.

✨ **Features**
- List-based movie suggestions
- Recommends movies based on matching genres
- Type `quit` (or similar) to exit the program

🛠️ **Technologies Used**
- python
